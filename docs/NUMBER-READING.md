# 数詞・単位の読みルールブック

字幕向け。形態素の桁読み（いちれいれい…）やアルファベット逐語読み（だぶりゅーえいち）を避け、
決定的な規則で固める。詳細実装は `src/number-unit-reading.js`。

## 1. 整数（整数部）

漢語の位取り読み。

| 例 | 読み |
|---|---|
| 1000 | せん |
| 210 | にひゃくじゅう |
| 320 | さんびゃくにじゅう |

- `0` 単独 → ぜろ（円など）。時刻の 0時だけ例外で「れい」
- カンマ（`1,000`）は桁区切りとして無視

## 2. 小数

学校数学・放送での一般則:

1. **小数点より左（整数部）** → 普通の整数読み  
2. **小数点「.」** → 「てん」  
3. **小数点より右** → **一桁ずつ**（位取り語を付けない）

| 例 | 読み |
|---|---|
| 12.8 | じゅうにてんはち |
| 3.14 | さんてんいちよん |
| 0.5 | れいてんご |
| 94.02 | きゅうじゅうよんてんれいに |

出典の考え方: 小数点以下は「十分の一・百分の一」と読まず数字だけ（森の学校／IMABI 等）。

**例外: ドットが2つ以上（各段1桁）** は小数ではなくカウントダウン／区切り列。

| 例 | 読み |
|---|---|
| 3.2.1 | スリーツーワン |
| 3. 2. 1 | スリーツーワン |

英数字カタカナ読み。通常の小数 `2.1`→`にてんいち` とは別扱いで、こちらを先に判定する。

## 3. 十＋助数詞の促音

助数詞が **カ・サ・タ・ハ（パ）行** で始まるとき、末尾の「じゅう」→「じゅっ」。

| 例 | 読み |
|---|---|
| 50% | ゴジュッパーセント |
| 50点 | ごじゅってん |
| 210円 | にひゃくじゅうえん（「えん」は母音始まり → 促音なし） |

現代語は主流の「じゅっ」を採用（伝統形「じっ」も可。NHK は両様）。

## 4. 欧文単位 → カタカナ語として読む

記号・略語は **アルファベット逐語ではなく単位名**。表記はカタカナ。

| 記号 | 読み |
|---|---|
| V | ボルト |
| A | アンペア |
| W | ワット |
| Ah | アンペアアワー |
| Wh | ワットアワー |
| kWh | キロワットアワー |
| kW | キロワット |
| Hz | ヘルツ |
| % | パーセント |
| km / m / kg … | キロメートル 等 |

例: `12.8V` → じゅうにてんはち＋ボルト → 表示はカタカナ化して **ジュウニテンハチボルト**  
例: 単独の `Wh` → **ワットアワー**（だぶりゅーえいち にしない）

## 5. 和語・漢語の助数詞 → ひらがな

時・分・円・人・点・倍など。特殊読みは表で持つ（1人→ひとり、4時→よじ 等）。

## 6. 処理順（実装）

1. 数字トークン列（小数点・カンマ・任意空白含む）を一塊に  
2. 直後の単位トークンがあれば結合  
3. 整数／小数ルールで数詞を生成し、単位接尾を付ける  
4. 欧文単位なら全体をカタカナ化して表示保持  

単位なしの 2 桁以上の整数も位取り読み（1000→せん）。

**表示:** 漢字を含まない数字系（`360` / `93%` / `3.2.1` / `12.8V` 等）はルビではなくホバーのツールチップ。`1人`・`0時` など漢字混じりは従来どおりルビ。

## 7. まだ薄いところ

- 分数（3/4）・序数
- 和語の日付（ついたち以外の複合）の網羅
- SI 接頭語の全組合せ（µA 等）
- 「にい・よん／しい」などリズムのための長音化はしない（標準短形）
