# プライバシーポリシー

最終更新: 2026-07-16  
対象: YT Furigana（Chrome 拡張および関連のローカル／任意サーバー機能）

本ポリシーは、本ソフトウェアの利用に関する個人情報・端末データの取り扱いを説明します。法的助言ではありません。

## 1. 運営

- リポジトリ: https://github.com/Blackphi6/yt-furigana-extension
- 連絡: GitHub Issues / リポジトリに記載の連絡手段

## 2. 収集するデータ（Free / 既定）

**既定（Free）では、開発者のサーバーへ字幕本文や学習データを送信しません。**

| データ | 保存場所 | 目的 |
|--------|----------|------|
| 設定（エンジン種別・URL など） | `chrome.storage.sync`（Google アカウント同期の範囲） | 設定の保持 |
| ライセンスキー・読み API キー | `chrome.storage.local`（端末内。Google 同期しない） | 認証 |
| ユーザーが選んだ読み辞書 | `chrome.storage.local` | ふりがな学習 |
| 学習ログ（字幕断片・ページ URL） | `chrome.storage.local` に**自動蓄積**（上限あり）。書き出しはユーザー操作時のみ | 辞書改善の素材 |

字幕テキストの解析は、原則としてユーザーのブラウザ内（Kuromoji / Sudachi 等）で行います。

### YouTube / TVer 上での読み取り

- 画面上の字幕 DOM を読み、ルビを重ねます。
- リモートエンジン利用時など、必要に応じて YouTube の字幕トラック（timedtext / player API）を取得することがあります。取得目的はふりがな付与のみです。コメント・視聴履歴・説明文などは収集しません。

## 3. 任意機能で送信されうるデータ

ユーザーが自ら有効化・接続した場合に限ります。

| 機能 | 送信先 | 内容 |
|------|--------|------|
| 読みAPI（BYO） | ユーザーが指定した URL | 字幕テキスト（選択時はトラック全行のこともあり）、個人読み辞書（`user_dict`） |
| Ollama | ユーザーのローカル Ollama | 変換用テキスト |
| Premium 辞書同期 | ユーザーが指定した同期サーバー | 読み辞書、ライセンスキー |
| Premium 共有辞書取得 | 同上 | 認証トークン |
| Premium 購入（サイト） | Stripe Checkout および読みエンジン側 | 決済は Stripe。メールは任意入力時のみ |

**第三者の公開 API を指定した場合、その事業者のプライバシーポリシーが適用されます。** 本プロジェクトは当該サービスの管理者ではありません。

## 4. 広告・トラッキング

本拡張の既定実装に、広告 SDK や行動トラッキングは含みません。

## 5. 子どもの利用

13 歳未満を対象としたサービスではありません。保護者の監督のもとでご利用ください。

## 6. データの削除

- 拡張の削除、または Chrome のサイトデータ／拡張データ削除により、端末内の学習・設定・キーは消去できます。
- Premium 同期サーバー上のデータは、当該サーバーの運用者（セルフホスト時は利用者自身）の手順に従ってください。
- Stripe 上の決済記録は Stripe およびカード会社のポリシーに従います。

## 7. 変更

本ポリシーは予告なく更新されることがあります。重要な変更はリポジトリの更新日で示します。
