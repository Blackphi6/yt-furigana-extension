YT Furigana

YouTube / TVer の日本語字幕にひらがなルビを表示

変換エンジン

サーバーURLは上の「プラン」欄で設定します。ローカル: npm run reading-engine → http://127.0.0.1:8765。
個人辞書は user_dict として送信されます。

上級者向け（ローカル LLM / Ollama）

自由生成LLMは本文を書き換えやすいため、通常は推奨しません。読みAPIやKuromojiの方が安定です。

推奨（軽い）: ollama pull gemma3:4b

プラン

Free

Free: ローカルふりがな・クリック学習は無制限（クラウド送信なし）。
Premium: 辞書クラウド同期・共有辞書・ホスト読みAPI。

ローカル検証用デモキー: ytfp_live_demo_key_001（要 npm run reading-engine）。

Premium を購入 · GitHub Sponsors で支援

自動学習

字幕の漢字を押すと読み候補が出ます。選ぶと端末内に学習されます。 Free ではクラウド送信しません。Premium の「辞書を同期」で端末間共有できます。 曖昧な読みのログも端末内に溜められます。書き出して data/learning/inbox.jsonl に置き、npm run learn で辞書を自動更新できます。

字幕が表示されない場合は、YouTubeの字幕設定で日本語を有効にしてください。
通常利用は Kuromoji（自己負担0・ローカル）を推奨します。
