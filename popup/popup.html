YT Furigana

YouTube / TVer の日本語字幕にひらがなルビを表示（非公式）

ふりがな

標準は端末内の Kuromoji（設定不要）。固有名詞辞書とクリック学習に対応しています。

上級者向け（エンジン切替）

通常は Kuromoji のままで大丈夫です。分割を強化したいときだけ Hybrid に切り替えてください。

プラン

Free

Free: ローカルふりがな・クリック学習は無制限（クラウド送信なし）。
Premium: 辞書クラウド同期・共有辞書。

ローカル検証用デモキー: ytfp_live_demo_key_001（要 npm run reading-engine）。

Premium を購入 · GitHub Sponsors で支援

自動学習

字幕の漢字を押すと読み候補が出ます。選ぶと端末内に学習されます。 Free ではクラウド送信しません。Premium の「辞書を同期」で端末間共有できます。 曖昧語のサンプル（字幕断片・ページ URL）も端末内に自動蓄積されます（上限あり・開発者サーバーへは送らない）。書き出して data/learning/inbox.jsonl に置き、npm run learn で辞書を自動更新できます。

字幕が表示されない場合は、YouTubeの字幕設定で日本語を有効にしてください。
通常利用は Kuromoji（端末内・設定不要）で十分です。
