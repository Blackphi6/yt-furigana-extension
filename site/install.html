インストール

非公式の Chrome 拡張です（Google / YouTube / TVer とは無関係）。 Free はすぐ使えます。Chrome Web Store 公開前は、以下のどちらかで追加してください。

A. リリース zip（おすすめ）

  1. 最新リリース から yt-furigana-extension.zip をダウンロード
  2. zip を展開する
  3. Chrome で chrome://extensions を開く
  4. 右上の「デベロッパーモード」をオン
  5. 「パッケージ化されていない拡張機能を読み込む」→ 展開したフォルダを選ぶ

B. ソースからビルド

  1. リポジトリ を clone
  2. npm install && npm run build
  3. 上記と同じく chrome://extensions からフォルダを読み込む

使い方

  1. YouTube または TVer で日本語字幕をオン
  2. 漢字の上にふりがなが付く
  3. 漢字を押すと読み候補 → 端末内に学習（Free）
