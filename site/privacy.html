1. 運営

  • リポジトリ: https://github.com/Blackphi6/yt-furigana-extension

2. 収集するデータ（Free / 既定）

既定（Free）では、開発者のサーバーへ字幕本文や学習データを送信しません。

  • 設定 → chrome.storage.sync（設定の保持）
  • ライセンスキー・読み API キー → chrome.storage.local（端末内。Google 同期しない）
  • ユーザー辞書 → chrome.storage.local（端末内）
  • 学習ログ（字幕断片・ページ URL）→ 端末内に自動蓄積（上限あり）。ポップアップでオプトアウト可。書き出しはユーザー操作時のみ

字幕テキストの解析は、原則としてユーザーのブラウザ内で行います。

YouTube / TVer では画面上の字幕 DOM を読みます。リモートエンジン利用時などは、ふりがな付与のため YouTube の字幕トラック（timedtext / player API）を取得することがあります。コメントや視聴履歴は収集しません。

3. 任意機能で送信されうるデータ

ユーザーが自ら有効化・接続した場合に限ります。

  • 読み API（BYO）: 指定 URL へ字幕テキスト・個人読み辞書（選択時はトラック全行のこともあり）
  • Ollama: ユーザーのローカル Ollama へ変換用テキスト
  • Premium 辞書同期 / 共有辞書: 指定サーバーへ辞書・ライセンスキー
  • Premium 購入（Stripe）: 決済は Stripe。ライセンス発行は読みエンジン側。メールは任意入力時のみ

第三者の公開 API を指定した場合、その事業者のプライバシーポリシーが適用されます。

4. 広告・トラッキング

本拡張の既定実装に、広告 SDK や行動トラッキングは含みません。

なお本 Web サイト（ポリシーページ等）は表示のため Google Fonts を読み込むことがあります（拡張本体とは別）。

5. 子どもの利用

13 歳未満を対象としたサービスではありません。

EEA / UK 等での追加権利（アクセス・消去等）については、既定データは端末内のみのため拡張の削除または学習ログ消去で対応できます。連絡先は GitHub Issues です。

6. データの削除

  • 拡張の削除、または Chrome の拡張データ削除で端末内データは消去できます。
  • ポップアップで学習ログ消去、および曖昧語サンプル自動蓄積のオフが可能です。
  • Premium 同期サーバー上のデータは、当該サーバー運用者の手順に従ってください。
  • Stripe 上の決済記録は Stripe およびカード会社のポリシーに従います。

7. 変更

本ポリシーは予告なく更新されることがあります。重要な変更はリポジトリの更新日で示します。
