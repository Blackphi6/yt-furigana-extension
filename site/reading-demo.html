仕組み（要約）

候補制約型の読み付け（候補生成 → 文脈で選択 → 低確信は辞書へ → ユーザー辞書が最優先）。 LLM に自由生成で読ませません。

  1. user_dict / trust（慣用句）
  2. UniDic + 同形異音ラティス（候補外禁止）
  3. ModernBERT rerank（あれば）／cue
  4. 閾値未満は base へフォールバック
