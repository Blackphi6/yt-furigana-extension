Privacy Policy

Last updated: 2026-07-22

This is not legal advice. Japanese version: privacy.html

1. Operator

Repository: GitHub. Contact: GitHub Issues.

2. Default (Free) data

We do not send caption body text to our servers by default.

  • Settings → chrome.storage.sync
  • License / API keys, reading dictionary, learning inbox → chrome.storage.local (learning inbox opt-out available)
  • Shared readings pack receive (default on, can disable): surface→reading only from the public reading API. No caption body.

On YouTube / TVer we read on-screen caption DOM only. Default playback does not call YouTube timedtext.

3. Optional sends

  • BYO reading API / Ollama: only if you configure them
  • Premium sync / server shared dict: user-configured server
  • Anonymous corrections (opt-in): surface, reading, short context only (no video URL)
  • Demo reading proposals (optional): surface→reading only into a review queue — not published to the Free pack until curated after heuristic / optional LLM review
  • Stripe checkout (site): payment via Stripe

Public API may rate-limit by IP. Opt-in votes and demo proposals use a non-reversible voter id derived from IP (not published in the pack).

4. Ads / tracking

No ad SDK or behavioral tracking in the extension.

5. Children

Not directed at children under 13.

6. Deletion

Uninstall the extension or clear extension storage. Disable shared pack receive in the popup. Server-hosted Premium data follows the server operator’s process.
